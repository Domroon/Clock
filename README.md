# Clock
![clock](https://user-images.githubusercontent.com/48067330/192119668-c8ff1a43-9c14-407d-bd2b-564a66d672cd.jpg)

## Installation

    cd Clock
    python -m venv venv
    ./venv/Scripts/activate
    pip install -r requirements.txt
    
## Start
    cd Clock
    ./venv/Scripts/activate
    python clock.py
